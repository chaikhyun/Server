# 실시간 알림 시스템

## 📌 프로젝트 개요
관리자 웹 서비스에 실시간 알림 기능을 추가하기 위한 백엔드 프로젝트입니다.

## 🛠 기술 스택
- Java 17
- Spring Boot 3.x
- Spring Security
- JPA (Hibernate)
- Redis (Pub/Sub)
- Server-Sent Events (SSE)
- MySQL
- Git & GitHub

## 🔍 주요 기능
- 인증 사용자 전용 알림 구독 (SSE 방식)
- Redis 기반 실시간 메시지 브로드캐스트
- 알림 구독 상태 관리 및 재연결 처리
- 알림 수신 여부 기록 및 읽음 처리
